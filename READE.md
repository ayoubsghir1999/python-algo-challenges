# 🐍 Python Algo Challenges

Collection de défis algorithmiques résolus en Python — du plus simple au plus corsé.  
Un algo par jour pour rester sharp et garder le GitHub actif. 💪

---

## 🎯 Objectif

Pratiquer la logique de programmation à travers des défis concrets :
affichage de patterns, manipulation de nombres, tri, récursion, et bien plus.  
Chaque script est **commenté** et **autonome** — lisible même sans contexte.

## 📁 Structure du repo

```
python-algo-challenges/
│
├── triangles/
```

## 🚀 Lancer un script

Aucune dépendance externe, Python 3.x suffit.

```bash
# Cloner le repo
git clone https://github.com/ton-user/python-algo-challenges.git
cd python-algo-challenges

# Lancer un défi
python triangles/triangle_pascal.py
```
